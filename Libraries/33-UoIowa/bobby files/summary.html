Accessibility Summary Report

Bobby Approval

Congratulations! These web pages contain no accessibility errors that Bobby can detect. There are, however, some checkpoints that an automatic program like Bobby cannot examine. These item(s) are presented below. You must be able to respond to these questions affirmatively in order to obtain Bobby Approval. If this summary report includes all the pages on your web site, you are entitled to use the Bobby Approved icon on your web site. A copy of the image (approved.gif) can be found in the images folder of the Bobby installation folder.

High priority items, manual check

Although these pages pass Bobby's criteria on items that it can check, there are other items that must be addressed in order to meet the minimum standard for accessibility as defined by the W3C. These 7 item(s) are presented below.

  1. Ensure that descriptions of dynamic content are updated with changes in content.
  2. Did you provide accessible alternatives to the information in scripts, applets, or objects?
  3. If this table contains data in rows and columns (i.e. a spreadsheet), have you identified headers for the table rows and columns?
  4. Provide alternative content for each SCRIPT that conveys important information or function.
  5. If any of the images on this page convey important information beyond what is in each images alternative text, add descriptive (D) links.
  6. If any of the images on this page convey important information beyond what is in each images alternative text, add a LONGDESC attribute.
  7. Make sure that style sheets transform gracefully.
