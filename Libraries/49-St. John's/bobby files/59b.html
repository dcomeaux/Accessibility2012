Tutorials

The following tutorials allow the user to master research skills in a self-paced, self-directed manner.

To view these tutorials, your computer must have Flash Player installed. [ Download a free copy of Flash player]

Finding Books

This page provides tutorials on using the SJU Library catalog, which is used to find books, videos, CDs, and other materials the library owns.

Finding Articles

Learn how to find magazine and journal articles using databases such as EBSCOhost, ProQuest Direct, Lexis-Nexis, and others!

Finding Reserve Materials

Learn how to find reserve materials, in both print and electronic form.

Information Literacy Tutorial

This is a comprehensive tutorial which covers all aspects of the research process, including: forming a thesis, finding and evaluating information, and citing sources and avoiding plagiarism.
