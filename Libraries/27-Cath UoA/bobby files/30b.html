    What is RefWorks?

    RefWorks is a Web-based bibliography and database manager that allows you to create your own personal database by importing references from text files or online databases and other sources. Use these references in writing papers and automatically format the paper and the bibliography in seconds with Write-N-Cite. You must create an account the first time you use RefWorks, and log into that account each time you return.

     

    How do I access RefWorks?

    Just like accessing other library databases, you can access RefWorks from the "Databases and More" drop down menu on the CUA Libraries' home page and selecting "RefWorks". Or click Here to go directly. If you are off campus, you then need to log into ALADIN using your CUA ID. Access to RefWorks is restricted to current students, faculty and staff of the University. Once you reach the RefWorks website, you will need to create or log into your personal account in RefWorks.

     

    How do I use RefWorks?

    Please view the following instruction guide and online tutorial for help in getting started.

    • RefWorks Quick Start Guide (.pdf)
    • RefWorks Online Tutorial
    • RefWorks -- Importing from databases instructions: Learn how to import records from a vendor database (e.g. EBSCOhost, Ovid, ProQuest, even Google Scholar) to your RefWorks account. [A .doc version of the instruction is also available.]

     

    What is Write-N-Cite? And how do I use it?

    Write-N-Cite is a small free download program provided by RefWorks, which enables you to access the abbreviated version of your RefWorks account while you are working in MS Word.

    There are 3 versions of the Write-N-Cite:

    • Version 2 is used for Windows
    • Version 2.5 is for Mac
    • Version III is for Windows Vista and Word 2007 users (it also works with XP and Word 2003). It allows working offline, which Version 2 does not.

    To download Write-N-Cite, log into your RefWorks account. On the top bar in RefWorks, click ?Tools?, and then select ?Write-N-Cite? from the drop down menu. On the next page, select the version to download, and then follow the directions on the screen.

    For detailed instructions on downloading and using Write-N-Cite, please follow the links on the download page, and see the RefWorks Online Tutorial.

    If you are using Write-N-Cite off campus, and don't want using the Group Code each time, you can load the WNC Proxy Configuration Utility by following these steps:

    For Windows users:

    1. From the Start Menu select Programs and RefWorks
    2. Select the WNC Proxy Configuration Utility
    3. Enter this URL:
      http://proxycu.wrlc.org/login?url=https://www.refworks.com/Refworks/?WNC=true
    4. Click OK.

    For Mac users:

    1. Start Write-N-Cite
    2. Select Preferences from the Write-N-Cite menu
    3. Select the Proxy option and enter this URL:
      http://proxycu.wrlc.org/login?url=https://www.refworks.com/Refworks/?WNC=true
    4. Restart Write-N-Cite for this proxy configuration to take effect.

     

    What is the Group Code? And how do I get it?

    The CUA RefWorks Group Code can be used as an alternative way to access CUA's RefWorks if you don't want to log into ALADIN. The Group Code will be updated periodically. For the current Group Code, please click here. (Off campus users will need to log in using your CUA ID to view the code.)

     

    What is RefWorks 2.0? How to toggle between the two versions?

    RefWorks 2.0 is the new user interface of RefWorks, which has many new easy-use features (read more about RefWorks 2.0). The old interface (aka "Classic") will be gone by June 2011. Before then, CUA users can freely toggle between the two versions:

    From Classic to RefWorks 2.0 From RefWorks 2.0 to Classic
    1. Log into your RefWorks account
    2. Click on the blue link "RefWorks 2.0" at the top right corner
    [image showing the switch link to RefWorks 2.0]
    3. Confirm your selection    		 1. In your RefWorks account, click on the first red link "RefWorks Classic" at the top
    [image showing the switch link to RefWorks Classic]
    2. Confirm your selection
    Please note: RefWorks will remember what interface that you have selected upon exiting your account.
