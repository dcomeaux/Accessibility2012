    Adding a Library Printer to Your Laptop

    Select Your Operating System

    • [-] Windows Vista
    • [-] Windows XP, 2000
    • [-] Windows 7
    • [-] Macintosh OS Leopard or Tiger
    • [-] Macintosh OS Lion

     

    Windows Vista

    Adding library printers to laptops via the wireless network in Microsoft Windows Vista:

    • From Control Panel, under "Hardware and Sound" select Printer.
    • Choose "Add Printer," (located near the top of the window)
    • Click "Add a network, wireless or Bluetooth printer".
    • The computer will search for available printers.
    • The IP addresses correspond with the various printers in the library:
    Gowan Nursing / Biology 136.242.48.210
    Mullen Information Desk 136.242.180.64
    Mullen MERIC Computer Lab 136.242.180.62
    Mullen 2nd Floor Hall - Copier 1 136.242.180.60
    Mullen 2nd Floor Hall - Copier 2 136.242.180.65
    Mullen Religious Studies South (314) 136.242.180.66
    Mullen Religious Studies North (316) 136.242.180.67
    Pangborn Engineering / Architecture 136.242.141.100
    Ward Music 136.242.21.33
    • Select a printer from the list.
    • Click "next".
    • Name the printer and check the box if you want this printer to set as your default printer.
    • Click "next."
    • Rename the Printer and click "next."
    • Select "Do Not Share" and click "next."
    • Select whether you want a Test Page printed, then click "next."
    • Click "Finish."

    Windows XP or 2000

    Obtaining the printer drivers:
    This step is not neccesary if you are using Windows XP.

    • Go to www.hp.com
    • Click on "Support and Drivers"
    • Click on "HP software and Drivers
    • Submit printer info. and download driver suitable for your Operating System (OS)*
      *If you are unsure as to what OS you are using, right-click "My Computer" and select "properties"

    Adding library printers to laptops via the wireless network in Microsoft Windows:

    • From Control Panel, choose Printers.
    • Choose "Add Printer," (for XP this may in the left frame)
    • Click "next" to bypass Microsoft page.
    • Select "Local Printer," and make sure the box below it is unchecked (Automatically detect and install my plug and play printer)
    • Select "Create a new port."
    • Select "Standard TCP/IP Port" from the drop-down menu.
    • Click "next" if the computer is connected to the network.
    • Add the IP address for the printer and click "next":

    Windows 7

    Adding library printers to laptops via the wireless network in Microsoft Windows 7:

    • Click the Start button.
    • From the Start Menu, choose "Devices and Printers."
    • Click "Add a printer" (near the top of the window).
    • Select "Add a local printer," and click Next.
    • Select "Create a new port."
    • Select "Standard TCP/IP Port" from the drop-down menu.
    • Click "Next" if the computer is connected to the wireless network.
    • Add the IP address for the printer and click "Next":

    Macintosh OS Leopard or Tiger

    Adding library printers to laptops via the wireless network in Mac OS Leopard or Mac OS Tiger:

    • Launch the Printer Setup Utility. You can find this by opening the Applications folder, then the Utilities folder.
    • Select the "Add" icon near the top of the screen.
    • Once the Printer Browser window is opened, make sure "IP Printer" is selected.
    • Make sure "Line Printer Daemon - LPD" is selected from the Protocol line.
    • Enter the IP address of the printer you wish to print from in the "Address" line.
    • The IP addresses correspond with the various printers in the library:
    Gowan Nursing / Biology 136.242.48.210
    Mullen Information Desk 136.242.180.64
    Mullen MERIC Computer Lab 136.242.180.62
    Mullen 2nd Floor Hall - Copier 1 136.242.180.60
    Mullen 2nd Floor Hall - Copier 2 136.242.180.65
    Mullen Religious Studies South (314) 136.242.180.66
    Mullen Religious Studies North (316) 136.242.180.67
    Pangborn Engineering / Architecture 136.242.141.100
    Ward Music 136.242.21.33
    • Click "Add".
    • Click "Continue".
    • The printer will appear in the Printer List.
    • Select the printer when printing a document.

    Macintosh OS Lion

    Adding library printers to laptops via the wireless network in Mac OS Lion:

    • Open System Preferences and then select Print &apm; Scan
    • Click on the plus sign ‘+’ on the left hand side of the window.
    • On the window that was just opened select ‘IP’ from the menu.
    • Make sure "Line Printer Daemon - LPD" is selected from the Protocol line.
    • Enter the IP address of the printer you wish to print from in the "Address" line.

    • If the above directions don’t work, try changing the Protocol line to one of the other options while keeping everything else the same.
