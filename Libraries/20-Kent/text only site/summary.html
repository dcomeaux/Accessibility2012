Bobby Approval

Congratulations! These web pages contain no accessibility errors that Bobby can detect. There are, however, some checkpoints that an automatic program like Bobby cannot examine. Although these pages pass Bobby's criteria on items that it can check, there are other items that must be addressed in order to meet the minimum standard for accessibility as defined by the W3C. These 5 item(s) are presented below.
