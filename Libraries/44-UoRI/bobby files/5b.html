Welcome.

Thank you for visiting the URI Library Skills Tutorial. Our hope is that by completing this tutorial, you will be acquainted with the tools that this library can provide to help you do your research and your assignments.

You will notice that this tutorial is broken up into different modules: Orientation, Finding Books, Finding Articles, and Feedback.

You can complete these modules consecutively, or if you prefer, you can jump around to the section that you need the most. Please remember, if you have any questions about using the library or doing research, be sure to ask a librarian at the reference desk.

If you are ready to begin, click continue below.
