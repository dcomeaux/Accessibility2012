Other Borrowers (Courtesy Cards)

Obtaining a Courtesy Card (a UIUC library card)

Courtesy Card Types

  • UIUC Courtesy Borrower Type LH (Visitors sponsored by an academic department; Members of UI President's Council; members of Allied Agencies; Holders of CIC ID Card or in state OCLC Reciprocal Faculty Borrowing Program Card; and K-12 Illinois teachers)
  • UIUC Courtesy Borrower Type LL (Non-UIUC affiliated state of Illinois residents with valid ID; persons holding UIUC Spouse IDs; UI alumni living in Illinois; Non-registered UIUC doctoral students living in Illinois; Extra Help workers and Research Park Employees)
  • Building Use Only Permits (Non-UIUC affiliated persons residing outside the state of Illinois with valid ID)
