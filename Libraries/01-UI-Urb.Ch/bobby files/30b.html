Searching for Reserves

Search for all Course Reserves within the Online Catalog

Use the above link to search for all course reserve materials by instructor or course. Because many of the University Library departmental libraries place materials on reserve, there may be multiple locations where items are on reserve. Searching within the library online catalog is the only way to view all reserves for a particular course, including print reserve materials, electronic reserves, and any media items.

Search by Course for Electronic Reserves only

This link takes you to a list of courses that have currently accessible electronic reserves. Print and media materials on reserve will not appear in these course specific lists. This link is accessible even when the library online catalog is experiencing difficulties. You may also want to view our E-Reserve Quick Problem-Solving Guide.
