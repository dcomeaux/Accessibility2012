Online Library Catalog

U of I Catalog

  • Library Catalog

    Find items by title, author, subject heading,  or keywords.  Includes all items found in the Classic catalog.  Note: no call number search and no limiting to UI unit libraries.

  • Classic Catalog: Quick Search

    Find items by titles, journal title, author, subject heading, call number, or keywords.

  • Classic Catalog: Advanced Search

    Use drop-down lists to combine keywords.

  • Classic Catalog: Course Reserves

    Search for course materials by instructor and course.

  • "My Library Account"

    Log in to renew materials, view pending Catalog requests and fines/fees owed, and more.   You must have an account in the catalog to use the new "My Account."  Use your 14-digit Library ID number to set up an account.

  • I-Share Catalog

    Find items in the joint catalog shared by U of I and 75 other academic libraries in the state of Illinois.

    To request items not found via I-Share, use  Interlibrary Loan.

  • Mobile Library Catalog

    A mobile interface for the library catalog.

  • WorldCat

    Find items not available via i-Share.

  • New Titles

    Browse the Library's recently cataloged new titles.
