Accessibility Summary Report

Bobby Approval

This collection of pages does not yet meet the requirements for items that Bobby can detect. In addition, there are a number of items that an automatic program like Bobby cannot examine. These 32 item(s) are presented below. You will need to be able to respond affirmatively to these items as well to obtain Bobby Approved status for these pages.

  1. Provide alternative text for all images.


High priority items, manual check

In addition to access errors on items that Bobby can check which are identified in the section following this one, there are other items that it is important to address in order to meet the minimum standard for accessibility as defined by the W3C. These 7 item(s) are presented below.

  1. Ensure that descriptions of dynamic content are updated with changes in content.
  2. If you cant figure out any other way to make a page accessible, construct an alternate version of the page which is accessible and has the same content.
  3. If this table contains data in rows and columns (i.e. a spreadsheet), have you identified headers for the table rows and columns?
  4. Provide alternative content for each SCRIPT that conveys important information or function.
  5. If any of the images on this page convey important information beyond what is in each images alternative text, add descriptive (D) links.
  6. If any of the images on this page convey important information beyond what is in each images alternative text, add a LONGDESC attribute.
  7. Make sure that style sheets transform gracefully.
