University Library System Student Job Opportunities

To apply:

 

* Download and print the online Student Employment Application Form (PDF)

* Complete the application and submit directly to the supervisor of the position


* Questions about a particular job should be directed to the supervisor listed

 

Students: A free shuttle service (PDF) runs between Hillman Library and the Library Resource Facility on Thomas Boulevard, the location of several of the jobs listed

 

Position: Information Systems is reviewing applications for part-time employment. Information Systems is responsible for the upkeep of computer hardware and software in the all ULS libraries. This position requires working between the ULS Library Resource Facility in Point Breeze (a FREE Shuttle is available between Hillman Library and Thomas Blvd.) as well as problem resolution on the Oakland campus. Must work shifts of no less than 4 consecutive hours NOT including travel time. The duties of this position(s) may include, but are not limited to:

  • PC hardware and software installation and configuration
  • Data entry in various applications
  • End-user training (minimal)
  • Documentation development
  • PC hardware and software troubleshooting
  • Computer hardware and software inventory
  • Database maintenance

We are looking for motivated individuals who are self starters, reliable and able to work independently with minimal direct supervision. The following skills are desirable:

  • Experience with Intel-based PCs and Windows XP/7
  • Organizational skills
  • Experience with all Microsoft products -  Microsoft Access, Excel, Word, PowerPoint, Outlook
  • Excellent diagnostic and troubleshooting abilities
  • Excellent communication skills
