Accessing Articles, Electronic Books, and Electronic Journals From Off Campus

CONNECT TO SECURE REMOTE ACCESS: https://sremote.pitt.edu

 

 

Off-campus access to most articles, ebooks, electronic journals, and databases is available only to current Pitt students, faculty, and staff*. How to connect?

Go to

https://sremote.pitt.edu

 

  • This service, known as Secure Remote Access, enables Pitt users to access online library journals and other restricted resources from any Internet connection through a Web interface.
  • More information and detailed instructions for the Secure Remote Access Service are available from the CSSD Documentation Web site.

 

 

Have a question? Contact Us

*Why only Pitt students, faculty, and staff? When we purchase access to an electronic resource, we sign a license agreement with the vendor. This license requires us to restrict access to current Pitt affiliates.
