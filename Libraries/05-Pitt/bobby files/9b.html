Borrow from other libraries

Borrow directly from other libraries using E-ZBorrow , get a Special Borrower Card to visit and borrow from other libraries like Carnegie Mellon, Duquesne University and Carnegie Library of Pittsburgh, or use Requests from Other Libraries (Interlibrary Loan).

To see which libraries worldwide own the item, check the WorldCat database.
