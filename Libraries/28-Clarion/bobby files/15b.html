Ask A Librarian

[Golden Eagle Mascot]

Reference services are available only when classes are in session.

Visit

  

Carlson Reference Desk on Level 1 (Clarion Campus)
Suhr Reference Desk (Venango Campus)

Call

  

Carlson (Clarion Campus): 814-393-2490 or 814-393-1841
Suhr Reference Desk (Venango Campus): 814-393-1242 ... or 
814-676-6591 extension 1242

E-mail

  

Complete the online form and one of our Reference Librarians will get back to you with an answer within 24 hours.

Live Chat

  

There's no software to download. No need to sign up for an account. Simply click a button to start a LIVE conversation with one of our Reference Librarians!
Chat Now...

Text

  

Send your questions via SMS text to our email address: libsupport@clarion.edu
Remember that SMS stands for Short Message Service. If the question (or answer) requires in-depth discussion, you probably should contact the Reference Librarian using one of the other options above.
Note: Standard messaging rates apply.
