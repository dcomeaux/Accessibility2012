Instructors, you can now request that books owned by the library be placed on reserve in the Undergraduate Library using the UL Reserve Tool in the library catalog. See details.


How to Search for Reserves

If you believe that your course reserves have already been processed and you are having trouble finding them in the search, try using the instructor name. Additionally, typing the 4-letter, 3-digit code should be done without a space (ex. PSYC101). If this does not work, your readings may still be in processing or may be available via paper reserve only.
