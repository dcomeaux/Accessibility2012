Library Widgets

This section of the University of Texas Libraries Web site lists a variety of widgets that you may find useful while conducting research.

What is a widget? Widgets bring the Libraries Web services to you. They include Web applications, simple add-ons for your Web browser, and other shortcuts that make researching a little easier. If you would like to learn more about widgets, watch our video demonstration or read this Newsweek article on the topic.

How do I get started? Each of the widgets we've listed on our site includes a description and installation instructions. Click on a widget below to learn more.

Search Widgets

CD/DVD Search Plugin for Firefox and IE What is it? An add-on to your Web browser that allows you to search the CDs and DVDs available in the University of Texas Libraries. Facebook App for the University of Texas Libraries What is it? An application you add to your Facebook account that allows you to search the University of Texas Libraries resources from within Facebook. Google Gadget for the University of Texas Libraries What is it? An application you can add to your personalized Google homepage, a Google Wave, Blackboard, or any other Web page. Links you directly to IM a Librarian service, the catalog, databases and more. Google Toolbar Button for the University of Texas Libraries What is it? A button you can add to your browser's Google Toolbar that provides an easy way to search the University of Texas Libraries Catalog. Law Library Search Plug-in What is it? A Web browser add-on that allows you to search the University of Texas Law Library's catalog, TALLONS. Library Catalog Search Plugin for Firefox and IE What is it? An add-on to your Web browser that allows you to search the University of Texas Libraries catalog. LibX Add-on for the University of Texas Libraries What is it? A Web browser add-on that gives you direct access to the University of Texas Libraries resources. Netvibes Widget for the University of Texas Libraries What is it? An application you can add to your personalized Netvibes homepage. Links you directly to IM a Librarian service, the catalog, databases and more. OpenURL Referrer What is it? A Web browser add-on that converts bibliographic citations to URL's. University of Texas Digital Repository Search Plug-in What is it? An add-on to your Web browser that allows you to search the University of Texas Digital Repository. WorldCat for iPhone What is it? An iPhone version of WorldCat.

Information Organizing Widgets

CiteMe Facebook App What is it? A Facebook app that creates formatted citations in APA, Chicago, Harvard, MLA, or Turabian style for books and other items found in WorldCat. Del.icio.us Web Browser Add-ons What is it? Web browser add-ons that make it easier to bookmark and organize web sites with http://del.icio.us. Flickr Uploadr What is it? A tool that enables you to easily upload many images to your Flickr account. NoodleBib What is it? An awesome tool that helps you create MLA- or APA-formatted bibliographies online. Save as a Word document to turn in with your paper. Readability What is it? A simple tool that removes the clutter around what you're reading on the Web. TextMarker What is it? A Firefox browser add-on that highlights text online within your browser window. Then copy and paste as you wish. Zoho Notebook What is it? A tool that helps you aggregate content from multiple sources or applications. Zotero What is it? A Firefox browser add-on that helps you collect, manage, and cite your research sources.

Collaboration Widgets

ClipMarks What is it? A free browser add-on that lets you clip text, images and videos from inside Web pages then save, blog, email and print to share your "clips" with others. Google Docs What is it? A web site that enables you to create documents, spreadsheets and presentations online. Google Sidewiki What is it? A free browser add-on that lets you contribute helpful information to any web page. PBWorks What is it? A free wiki tool that is useful for everyone (even computer beginners).
