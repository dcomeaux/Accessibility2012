Using the Libraries

Library locations (Map of library locations on campus).

Phone numbers. For general information call 301-405-0800.

Borrowing material: Alumni can borrow from the UM Libraries if they are either current members of the UM Alumni Association or the Friends of the Libraries Borrowers Program. For policies and borrowing information see Borrowing Materials from UM Libraries.

Library computers. UM Alumni are invited to use public library computers, but first you must obtain a guest account to access online library resources. Apply at any campus library Circulation Desk; photo I.D. is required. Users conducting research are given first priority, so personal e-mail and other non-academic online activities must be kept to a minimum. Users are expected to abide by the University’s Acceptable Use Policy for information technology resources.

Access to the Internet Using Laptops. Only UM College Park students, faculty, and staff are permitted to access the Internet using their laptops on campus. UM Alumni may not use their laptops to get Internet access. UM Alumni are invited to use public library computers, but first you must obtain a guest account. Apply at any campus library Circulation Desk; photo I.D. is required.

Accessing the Catalog and Databases

  • Catalog:
    • On campus: UM Alumni are invited to use public library computers to search the catalog, but first you must obtain a guest account.
    • Off campus: Anyone can search the Libraries' Catalog off campus, regardless of affiliation with UM.
  • Databases:
    • On campus: UM Alumni are invited to use public library computers to access the databases, but first you must obtain a guest account.
    • Off campus: Only current students, faculty and staff can access University of Maryland Library on-line databases from a remote location.
