Accès hors campus aux ressources électroniques (Proxy)

La majorité des ressources électroniques (périodiques, bases de données et données numériques) offertes par les bibliothèques de l'Université de Montréal consistent en des abonnements payants régis par des contrats de licences et dont l'utilisation est réservée à la communauté universitaire de l'Université de Montréal. Le Proxy permet d'accéder à ces ressources de l'extérieur du campus.

Préalables essentiels pour établir la connexion au Proxy

  • Être étudiant ou membre du personnel de l’Université de Montréal
  • Avoir un code d'accès DGTIC et un UNIP/mot de passe
  • Utiliser un navigateur Web compatible et configuré pour le Proxy (voir onglets plus haut)

En cas de problème

  • Vérifier la configuration de votre navigateur internet (voir onglets plus haut)
  • Tester la configuration à l'adresse : http://proxy.umontreal.ca:81
  • Vérifier si une panne ou un arrêt de service est signalé par la DGTIC
  • Les utilisateurs de Windows Vista ou 7 ainsi qu'Internet Explorer doivent écrire SIM\ devant leur nom d'utilisateur (exemple : SIM\p1234567) en prenant soin d'utiliser la barre oblique qui penche vers la gauche: \. Si SIM\ n'est pas inscrit devant votre nom d'utilisateur, vous devrez cliquer sur Utiliser un autre compte et inscrire les informations correctement
  • Vider la mémoire cache de votre navigateur internet
    • Internet Explorer : Menu Outils (ou icône engrenage), Options Internet, onglet Général, bouton Supprimer, cocher Fichiers Internet temporaires,cliquer sur le bouton Supprimer. Redémarrer Internet Explorer
    • Firefox (PC) : Menu Firefox, Options, Avancé, Vider le cache maintenant. Redémarrer Firefox
    • Firefox (Mac) : Menu Outils, Supprimer l’historique récent…, Intervalle à effacer tout, dans Détails cocher Cache, cliquer Effacer maintenant. Redémarrer Firefox
    • Safari (mac) : Cliquer sur le menu Safari, sélectionner Vider le cache, cliquer sur Vider. Redémarrer Safari
    • Safari (Ipad, Iphone, Ipod) : Réglages, Safari, Effacer cookies et données, Effacer cookies et données. Redémarrer Safari
    • Chrome (PC) : Icône clé à molette, Options, Options avancées, Effacer les données de navigation..., Effacer les éléments datant de n'importe quand, cocher Vider le cache, cliquer Effacer les données de navigation. Redémarrer Chrome
    • Chrome (mac) : Icône clé à molette, Outils, Effacer les données de navigation, Effacer les éléments datant de n'importe quand, cocher Vider le cache, cliquer Effacer les données de navigation. Redémarrer Chrome
  • Désactiver tout autre logiciel ou barre d'outils avec bloqueur popup telle la barre d’outils Google
  • Avec Windows 7 et Firefox, il se peut que l'on demande de s'authentifier au Proxy sur des pages web sans lien avec les ressources électroniques. Pour éviter ce problème, enlever le crochet présent dans la configuration du Proxy
  • En dernier recours, réinitialiser les paramètres de votre navigateur internet
    Attention! Cause la perte des données de navigation personnelles ainsi que la désactivation des barres d'outils et compléments
    • Internet Explorer : Menu Outils (ou icône engrenage), Options Internet , onglet Avancé, bouton Réinitialiser, cocher Supprimer les paramètres personnels et cliquer sur Réinitialiser. Redémarrer Internet Explorer
    • Firefox (PC) : Ouvrir Firefox en maintenant la touche Majuscule. Tout cocher et cliquer sur Effectuer les changements et redémarrer
    • Firefox (Mac) : Ouvrir Firefox en maintenant la touche Alt. Tout cocher et cliquer sur Effectuer les changements et redémarrer
    • Safari (mac) : Cliquer sur le menu Safari, sélectionner Réinitialiser Safari et cliquer sur Réinitialiser
    • Chrome (PC) : Attention! Les favoris seront aussi supprimés
      Renommer le dossier "Default" situé dans le répertoire C:\Users\nom-utilisateur\AppData\Local\Google\Chrome\User Data. Redémarrer Chrome
    • Chrome (mac) : Attention! Les favoris seront aussi supprimés
      À partir du bureau (Finder), cliquer sur le menu Aller, Aller au dossier... puis taper ~/Library/Application Support/Google et cliquer sur Aller. Renommer le dossier Default. Redémarrer Chrome
