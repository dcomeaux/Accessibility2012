     Introducing EndNote

  • EndNote is an online search tool - it provides a simple way to search online bibliographic databases and retrieve the references directly into EndNote. It can also import data files saved from a variety of online services and library databases.
  • EndNote is a reference and image database - it specializes in storing, managing, and searching for bibliographic references in your private reference library.
  • EndNote is a bibliography and manuscript maker - it formats citations, figures, and tables in Microsoft Word with the Cite While You Write feature.

     EndNote Web delivers tools to:

  • Search online resources
  • Save ISI Web of Knowledge SM records directly to an online library
  • Collect and organize references
  • Format citations and footnotes or a bibliography

       New Features on EndNote x5 (Online Video)

  • Add and move file attachments between the computer and Web—requires registering with EndNote X5
    • Attach files to an EndNote Web record for flexibility in collecting reference information.
    • Store up to 25,000 references on the Web with 1GB for file attachments.
    • Transfer up to 5,000 references in each session.
  • Update references automatically
    • Let EndNote search online and present updated record information.
    • Update empty fields or replace all fields.
  • View and annotate PDF files on the main library window 
    • Navigate to other attachments easily.
    • Annotate a PDF with highlights and/or comments.
  • Cite While You Write™ New Options
    • Save time with conditional formatting in journal styles—substitute data if a field is empty.
    • Select a new citation style—Author (Year)—in Microsoft® Word.
  • Compare duplicate references quickly with auto-highlights and locked scrolling.
  • Cite While You Write in over 5,000 journal styles!

       New Features on EndNote x4 (Online Video)

  • Import and Search PDFs
    • Create records from PDF files easily—EndNote extracts the DOI bibliographic data and attaches the PDF to the record.
    • Import one file or a folder of PDFs into an EndNote library.
    • Search the contents of imported PDFs.
  • Collaborate more with EndNote Web
    • Transfer up to 10,000 references between the Web and desktop.
    • Manage personal publication lists for the free ResearcherID author community.
    • Sort reference lists by Times Cited, Date Added to Library and Date Modified.
  • Use new Cite While You Write™ Features
    • Cite While You Write in Word 2010.
    • Link in-text citations to the bibliography automatically in Microsoft® Word.
    • Format footnotes using customizable short forms.
    • Meet the complete APA 6th style requirements. 
    • View a citation report of references cited in a Word document with a new auto-group in the EndNote library.
  • Create new groups by comparing, combining and suppressing existing groups.
  • Edit references when comparing duplicates.
  • Modify references easily in the new Quick Edit tab on the main library window.

    ResearcherID Web Services

    • Managing a ResearcherID profile is easier than ever with EndNote. Now users can transfer references to the ResearcherID group presented in EndNote Web and update their publication list automatically. This free author community enables researchers to manage their own profile, eliminate author misidentification, search for collaborators and more. 
    • Users can use ResearcherID Web Services to highlight achievements and measure performance. Help busy researchers and increase NCCU’s visibility by mass creating profiles and uploading publication data. Learn More
