Plugins

The plugins help you access Western Libraries resources from wherever you are on the web.

Browser add-ons

LibX is a browser add-on for Firefox and Internet Explorer. It provides direct access to Western Libraries' resources, regardless of where you are on the web. Read more....

Search OED Online from your web browser. Just highlight a word on a web page and then click on the OED button to find the word's definition. If you want to use this at home, just log in to off-campus access before you install the button.

Simplify Off-Campus Access

Simplify off-campus access to journals and databases by installing the EZProxy button or using the EZProxy URL Generator to create bookmarks for journals and databases that will work at home.

If you use software to manage accessing databases or online journals, and if it supports accessing journals via a proxy server, the base URL to connect to Western Libraries proxy server is

https://www.lib.uwo.ca/cgi-bin/ezpauthn.cgi

RefWorks Tools

Add RefGrab-It to your browser to capture bibliographic information from web pages and import that data into your RefWorks account. RefGrab-It looks for information on web pages and if an ISBN number, PubMed ID or DOI exist on the web page, RefGrab-It will automatically search various web resources to get supplemental information that may be of interest to you.

OpenURL: Adding "Get it @ Western" to EndNote or Other Software

If you use EndNote, Reference Manager, or many other bibliography software packages, it is possible to add "Get it @ Western" links to your software using its OpenURL configuration option. To link directly from your bibliography software to article fulltext, set the software's OpenURL configuration to

http://sfx.scholarsportal.info/western

Want more?

Tell us what you're after and we'll see what we can do. Future plugins may include applications for Facebook, gadgets for Google Personalized homepage, and more.
