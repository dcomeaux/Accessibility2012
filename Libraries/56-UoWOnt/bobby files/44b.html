Library Computing

Logging In

Current students, faculty and staff of Western and the Affiliated University Colleges may access library computers by logging in to a public PC or a Sun Ray terminal using their Western Identity account username and password.

Western Alumni and other walk-in guests must present photo ID at a service desk to obtain a temporary username and password, which allows on-campus access to most online resources and the Internet. This temporary login can be used only on Sun Ray terminals at any Western Libraries’ location, and is valid until library closes on the day of issue.

Hardware and Software

Western Libraries provides access to public PCs as well as Sun Ray terminals. The software available on these workstations may vary from one location to another but includes a web browser for Internet access.

Public PCs provide access to Microsoft Office 2010. For more information about using our public PCs please visit our help page.

On Sun Ray terminals, documents may be opened and edited only from email attachments or links on a web page. These are accessible in OpenOffice, a type of office productivity software which is compatible with Microsoft Office. For information on using OpenOffice please visit our help page.

It is not possible to install software/applications on any public PC or Sun Ray terminal. Public PCs are also unable to run individual CD-ROM or DVD discs. Burning CDs/DVDs is not a service provided by Western Libraries.

Wireless

For information on using Wireless at Western Libraries, including printing information, please visit our help page.

USB Devices

Support for USB devices (flash drives) is available on public PCs. Open “My Computer” on the Desktop for access to USB directory and files. For more information please see our help page.

USB access is not available on Sun Ray terminals.

Printing

For information on printing at Western Libraries please visit our help page.
