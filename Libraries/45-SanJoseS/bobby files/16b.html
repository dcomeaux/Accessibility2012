San Jose State University eBooks

What's an ebook?

An ebook is an electronic book that can be read by using a personal computer or other digital device.

The Collections

SJSU Library and San Jose Public Library (SJPL) maintain separate ebook collections. The collections are also searchable in the library catalog. The University Library ebook collections are available to the SJSU community 24/7. SJSU users will need either their Tower card or their library card to access the SJSU collections off campus. All users need a SJPL library card to access the San Jose Public Library collections remotely. San Jose Public Library customers may access the University Library electronic resources only while in the King library.
